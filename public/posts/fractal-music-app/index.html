フラクタル音楽とは何か#

自然界には、海岸線・雲・樹木の枝分かれなど、**自己相似性（フラクタル構造）**を持つ形が無数に存在します。この自己相似性は空間だけでなく、時間変化にも現れます。

自然界のランダムな変動が
1/f ノイズ や 1/f² ノイズ といった周波数特性を持つことが示されています。

  • 白色ノイズ（f⁰） …… 完全なランダム
  • 1/f ノイズ（f⁻¹） …… 自然界に最も多いゆらぎ
  • ブラウン運動（1/f²） …… 変化のゆっくりした揺らぎ

これらは、過去と現在がゆるく相関しながら変化する “フラクタル的時間構造” を示すものです。

音楽も “1/f” のフラクタル構造を持っている#

興味深いことに、ほぼすべての音楽のメロディ変動は 1/f ノイズに非常に近いという研究結果があります。

  • クラシック
  • ジャズ
  • 民族音楽
  • ポップス

どのジャンルであっても、メロディの高さ変化を周波数解析すると
1/f に近いスペクトルが現れます。

つまり、

人が「音楽的だ」と感じるメロディは、1/f のランダム性と規則性のバランスの上に成り立っている。

  • ランダムすぎる白色ノイズ
  • ゆっくり変化しすぎる 1/f²
    の“ちょうど中間”にあるのが 1/f であり、人間の感性に一番フィットします。

なぜフラクタルから音楽が作れるのか#

1/f ノイズの性質を理解すると、

フラクタル構造を持つ信号そのものが「音楽の骨格」として使える
ことがわかります。

音楽は本質的にフラクタルであり、
だからこそ、

フラクタル数学を使って音楽を生成できる。

今回のフラクタル音楽生成アプリでは、

  1. 周波数領域で 1/f などのスペクトルを作り
  2. 逆フーリエ変換で時間領域へ戻し
  3. Web Audio API でリアルタイム再生

という仕組みを使って ホワイトノイズ／1/f ノイズ／1/f² ノイズの変動に対応する音響を生成しています。

このアプローチにより、
**自然な揺らぎ・ランダム性・規則性が混在した「フラクタル音楽」**を体験できます。

フーリエフィルタリング法#

このアプリでは、フーリエフィルタリング法を用いてフラクタルな音響信号を生成しています。

仕組み#

  1. 周波数領域での生成: 音のスペクトルをフラクタルパターンで構築
  2. 逆フーリエ変換: 周波数領域から時間領域へ変換
  3. 音響合成: Web Audio APIを使用してリアルタイムに再生

この手法により、自然でありながら予測不可能な、興味深い音楽パターンが生成されます。
