    フラクタル音楽生成アプリを作りました

    フラクタル音楽生成アプリ フーリエフィルタリング法を使用してフラクタルな音楽を生成するWebアプリケーションを作成しました。 アプリを試す → フラクタル音楽とは何か 自然界には、海岸線・雲・樹木の枝分かれなど、**自己相似性（フラクタル構造）**を持つ形が無数に存在します。この自己相似性は空間だけでなく、時間変化にも現れます。 自然界のランダムな変動が 1/f ノイズ や 1/f² ノイズ といった周波数特性を持つことが示されています。 白色ノイズ（f⁰） …… 完全なランダム 1/f ノイズ（f⁻¹） …… 自然界に最も多いゆらぎ ブラウン運動（1/f²） …… 変化のゆっくりした揺らぎ これらは、過去と現在がゆるく相関しながら変化する “フラクタル的時間構造” を示すものです。 音楽も “1/f” のフラクタル構造を持っている 興味深いことに、ほぼすべての音楽のメロディ変動は 1/f ノイズに非常に近いという研究結果があります。 クラシック ジャズ 民族音楽 ポップス どのジャンルであっても、メロディの高さ変化を周波数解析すると 1/f に近いスペクトルが現れます。 つまり、 人が「音楽的だ」と感じるメロディは、1/f のランダム性と規則性のバランスの上に成り立っている。 ランダムすぎる白色ノイズ ゆっくり変化しすぎる 1/f² の“ちょうど中間”にあるのが 1/f であり、人間の感性に一番フィットします。 なぜフラクタルから音楽が作れるのか 1/f ノイズの性質を理解すると、 フラクタル構造を持つ信号そのものが「音楽の骨格」として使える ことがわかります。 音楽は本質的にフラクタルであり、 だからこそ、 フラクタル数学を使って音楽を生成できる。 今回のフラクタル音楽生成アプリでは、 周波数領域で 1/f などのスペクトルを作り 逆フーリエ変換で時間領域へ戻し Web Audio API でリアルタイム再生 という仕組みを使って ホワイトノイズ／1/f ノイズ／1/f² ノイズの変動に対応する音響を生成しています。 このアプローチにより、 **自然な揺らぎ・ランダム性・規則性が混在した「フラクタル音楽」**を体験できます。 ...

    マークダウンで記事を書くメリット

    マークダウンとは マークダウン（Markdown）は、シンプルな記法で文書を書くための軽量マークアップ言語です。2004年にJohn Gruberによって開発されました。 マークダウンの基本記法 見出し # 見出し1 ## 見出し2 ### 見出し3 強調 太字は**太字**、イタリックは*イタリック*で表現します。 リスト 順序なしリスト 項目1 項目2 サブ項目2-1 サブ項目2-2 項目3 順序付きリスト 最初の項目 次の項目 最後の項目 リンクと画像 リンクテキスト [リンクテキスト](https://example.com) ![画像の説明](image.png) コードブロック プログラミングコードは、バッククォート3つで囲みます： function hello() { console.log("Hello, World!"); } マークダウンのメリット 1. プレーンテキストで管理できる .mdファイルは単なるテキストファイルなので、どんなエディタでも開けます。Gitなどのバージョン管理システムとの相性も抜群です。 2. 書くことに集中できる 複雑な書式設定に気を取られることなく、コンテンツの執筆に集中できます。 3. 可搬性が高い マークダウンファイルはHTML、PDF、Wordなど様々な形式に変換できます。 4. 学習コストが低い 基本的な記法は15分もあれば習得できます。 人気のマークダウンエディタ VSCode - 拡張機能が豊富 Typora - リアルタイムプレビュー Obsidian - ナレッジベース構築に最適 iA Writer - シンプルで美しいUI まとめ マークダウンは、技術文書、ブログ記事、メモ、ドキュメンテーションなど、あらゆる文書作成に活用できます。一度覚えれば一生使えるスキルです！

    Hugoを使った静的サイト構築入門

    Hugoとは Hugoは、Go言語で書かれた高速な静的サイトジェネレーターです。マークダウンファイルから美しいWebサイトを数秒で生成できます。 Hugoの特徴 1. 圧倒的な速度 Hugoは数千ページのサイトでも1秒以内にビルドできるほど高速です。これは他の静的サイトジェネレーターと比較しても圧倒的な速さです。 2. シンプルなセットアップ 単一のバイナリファイルとして配布されているため、依存関係の管理が不要です。インストールも非常に簡単です。 # macOSの場合 brew install hugo # バージョン確認 hugo version 3. 豊富なテーマ 300以上のテーマが用意されており、好みのデザインをすぐに適用できます。今回使用しているPaperModもその一つです。 基本的な使い方 新しいサイトの作成 hugo new site my-site cd my-site テーマのインストール git init git submodule add https://github.com/adityatelange/hugo-PaperMod themes/PaperMod 記事の作成 hugo new posts/my-first-post.md ローカルサーバーの起動 hugo server -D ブラウザで http://localhost:1313 にアクセスすると、サイトがライブプレビューされます。 まとめ Hugoは以下のようなサイトに最適です： 個人ブログ ポートフォリオサイト ドキュメントサイト 企業のコーポレートサイト マークダウンで記事を書くだけで、美しく高速なWebサイトが構築できます。ぜひ試してみてください！
