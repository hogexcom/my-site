    Picture Description App#

    英語のスピーキング練習として、画像を見てその状況を英語で説明する「Picture Description」の練習用Webアプリを作成しました。

    アプリを試す →

    できること#

    • ランダムに表示される画像を見て、その状況を英語で説明する練習ができます。
    • “Show Answer” ボタンを押すと、その画像に対応する英語の例文が表示されます。
    • “Random” ボタンでランダムに別の画像へ切り替えたり、“Prev” / “Next” ボタンで順にめくったりできます。

    シンプルながらも、毎日の英語学習のルーチンとして使えるように設計しました。
